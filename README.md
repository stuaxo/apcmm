APCMM
=====

APC Mini API and emulator.

(Fairly unfinished and experimental).

Install
=======

Update setuptools and friends
-----------------------------

```
$ python -mpip install -U pip setuptools wheel
```

Install
-------

```
$ pip install -r requirements.txt

$ python setup.py install
```


Run Emulator
------------
```
$ apcmm
```
